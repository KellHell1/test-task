Run:

    1.Docker, docker-compose and git are required

    2.Clone this repository to separate folder

    3.Execute ./run.sh in your terminal

For start frontend:
    
    1.npm install

    2.npm run dev

For test running:

    1. Entering in php container-> docker-compose exec php sh
    
    2. ./vendor/bin/phpunit tests


Path to main page:
    
    localhost:8081
